# VoiceDraw 设计文档

> XEngineer 第四批 · 题目二：AI 语音绘图工具

## 1. 项目概述

VoiceDraw 是一款**纯语音控制**的 Web 绘图工具。用户通过麦克风说出自然语言指令，系统在画布上绘制、修改图形。进入「连续语音模式」后，**无需鼠标和键盘**即可完成 Demo 全流程。

---

## 4. 技术架构

```
麦克风 → Web Speech API（浏览器免费 ASR）
              ↓
        规则指令解析 commands.js
              ↓
        画布引擎 canvas-engine.js → HTML5 Canvas
              ↓
        语音反馈 Web Speech Synthesis（可选）
```

**为何不用大模型：** 指令域固定、延迟低、Demo 稳定、**运营成本为 0**。

## 5. 成本说明

| 方案 | 成本 | 采用 |
|------|------|------|
| Web Speech API | 免费 | ✅ |
| 云端 ASR | 按量 | ❌ |
| GPT 语义解析 | 按 token | ❌ |

后续可对「解析失败」的句子按需调用轻量 LLM，预计每会话 < 0.01 元。

## 6. Demo 脚本（3 分钟）

1. 介绍：纯语音绘图，第四批题目二
2. 点「开启连续语音」或说「开始监听」
3. 依次说：**画一个红色的圆** → **大一点** → **右移** → **画蓝色矩形** → **画五角星** → **撤销** → **导出图片**
4. 展示右侧日志与下载的 PNG

## 7. 测试

1. Chrome/Edge 打开 `index.html`，允许麦克风
2. 说「开始监听」进入连续模式
3. 按 Demo 脚本操作，检查画布与日志
